# Team Collaboration Plan

## Branch Workflow
- `main` is the stable branch.
- Feature branches:
  - `feature/lambda-api`
  - `feature/terraform-infra`
  - `docs/slo-plan`
- All changes go through Pull Requests.

## Responsibilities
- **Sai Krishna:** Lambda functions, API design, README, documentation.
- **Sai Vardhan:** Terraform IaC, monitoring plan, IAM/policies, architecture diagrams.

## Review Process
- Each PR must be reviewed by the other teammate before merging.
- No secrets or keys will be committed.

## Tools
- GitHub for version control
- AWS console for cloud resources
- Terraform for IaC
- Postman/curl for API testing
